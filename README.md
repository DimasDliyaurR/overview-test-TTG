# Cara menjalankan kode berikut

## Soal 1

### **Tech Stack** 
	- HTML
	- JavaScript
	- CSS

### Cara penggunaan :
1. Melalui editor
	- Buka editor seperti sublime atau vscode 
	- Buka folder Soal 1
	- Buka file **`index.html`**
	- Klik kanan pada kode pilih opsi "open in default browser" atau "open in other browser"
2.  Melalui browser:
	- Buka browser
	- klik "ctrl + o", lalu pilih **`index.html`** di soal 1
3. Melalui folder manager
	- Buka folder soal 1
	- klik file **`index`**
